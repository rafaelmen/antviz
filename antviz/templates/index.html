AntViz Data Visualization

AntViz is a web application dedicated to the exploration of ant data stored in the AntWeb.org database. The visualizations on AntViz help describe the vast diversity of ant data available in the database.

The application is built using Python and the Bokeh visualization library. View all project code on GitHub.
